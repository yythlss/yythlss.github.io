随机事件与概率
Random Events and Probability
▼
1.1 随机事件与样本空间

定义随机试验

对随机现象进行的观测或实验统称为随机试验，需满足三个条件：

  • 在相同条件下可以重复进行
  • 每次试验的可能结果不止一个，且能事先明确所有可能结果
  • 每次试验前不能确定哪一个结果会出现，但总会恰好是其中一个

定义样本空间与事件

  • 样本空间 Ω：随机试验所有可能结果的集合
  • 样本点 ω：样本空间中的元素
  • 样本空间分类：离散样本空间（样本点有限或可列个）vs 连续样本空间（不可列无限个）
  • 随机事件：样本空间的子集。基本事件（不可再分）、复合事件（由基本事件组合）
  • 必然事件：Ω；不可能事件：∅
  • 事件 A 发生 ⟺ A 中某个样本点出现

性质事件间的关系与运算

关系：

  • 包含：A ⊂ B（A发生则B必发生）
  • 相等：A = B（A ⊂ B 且 B ⊂ A）
  • 互斥（互不相容）：A ∩ B = ∅
  • 对立（互逆）：A ∩ B = ∅ 且 A ∪ B = Ω，记 B = Ā

运算律：

  • 交换律：A ∪ B = B ∪ A，A ∩ B = B ∩ A
  • 结合律：(A ∪ B) ∪ C = A ∪ (B ∪ C)
  • 分配律：(A ∪ B)C = AC ∪ BC
  • 对偶律（De Morgan）：
    A∪B 的补 = Ā ∩ B̄
    A∩B 的补 = Ā ∪ B̄
⚠️ 易混淆：对立一定互斥，但互斥不一定对立。对立要求并集为全集。对立事件唯一，互斥事件可以有多个。

例题事件表示练习

设 A, B, C 为三个随机事件：

  • 仅 A 出现，B, C 不出现：ABC̄ 的补中取 A∩B̄∩C̄
  • 三个事件至少有一个出现：A ∪ B ∪ C
  • 不多于一个事件出现：ĀB̄C̄ ∪ ĀB̄C ∪ ĀBC̄ ∪ AB̄C̄
  • 至少有两个出现：AB ∪ AC ∪ BC
1.2 古典概型

定义古典概型条件与公式

满足：① 样本空间有限；② 每个样本点等可能出现。

古典概型公式 P(A) = A 包含的样本点数 / Ω 中样本点总数 = k / n

计数原理：

  • 加法原理：完成一件事有 n 类方法，第 i 类有 mᵢ 种 → 共 Σmᵢ 种
  • 乘法原理：完成一件事需 n 步，第 i 步有 mᵢ 种 → 共 Πmᵢ 种
  • 排列：P(n,k) = n! / (n-k)!（有序）
  • 组合：C(n,k) = n! / [k!(n-k)!]（无序）
解题步骤：设事件 → 找样本点 → 求概率

经典例题

  • 分房问题（生日问题）：n 个人，365 天，至少两人生日相同的概率。当 n=23 时概率已超 50%
  • 匹配问题（信纸装信封）：n 封信随机装入 n 个信封，至少一封匹配的概率 → 1 - 1/2! + 1/3! - … + (-1)^(n+1)/n!
  • 抽样问题：有放回 vs 无放回，当总体大而抽样少时两者近似相等
1.3 几何概型

定义几何概率

样本点等可能地落入空间某区域，通过几何度量（长度/面积/体积）计算概率：

几何概型公式 P(A) = A 对应区域的度量 / Ω 对应区域的度量

经典例题

  • 会面问题：甲乙约在 [0,T] 内会面，先到者等 t 后离去。P(会面) = 1 - (T-t)²/T²
  • Buffon 投针问题：间距为 a 的平行线，长 b (< a) 的针，P(相交) = 2b/(πa)
1.4 概率的公理化定义

定义事件域与概率空间

事件域 F（σ 域）满足：① Ω ∈ F；② 若 A ∈ F 则 Ā ∈ F；③ 若 Aᵢ ∈ F 则 ∪Aᵢ ∈ F

概率 P 是 F 上的集合函数，满足三条公理：

  • 非负性：∀A ∈ F，P(A) ≥ 0
  • 规范性：P(Ω) = 1
  • 可列可加性：对两两互斥的可列事件 A₁, A₂, …，P(∪Aᵢ) = ΣP(Aᵢ)

(Ω, F, P) 称为概率空间。

1.5 概率的性质

性质常用概率性质

  • P(∅) = 0
  • 有限可加性：互斥事件之并的概率 = 概率之和
  • 对立事件：P(Ā) = 1 - P(A)
  • 减法公式：P(A - B) = P(A) - P(AB)
  • 加法公式：P(A ∪ B) = P(A) + P(B) - P(AB)
  • 一般加法公式（多除少补）：P(A∪B∪C) = P(A)+P(B)+P(C) - P(AB)-P(AC)-P(BC) + P(ABC)
  • 单调性：若 A ⊂ B，则 P(A) ≤ P(B)
  • 连续性：单调增序列 Aₙ ↑ A → P(Aₙ) → P(A)；单调减序列类似
技巧：求"至少一个"的概率常转化为求对立事件（"一个都没有"）。P(AB) 的范围：max(0, P(A)+P(B)-1) ≤ P(AB) ≤ min(P(A), P(B))

条件概率和独立性
Conditional Probability and Independence
▼
2.1 条件概率与乘法公式

定义条件概率

设 P(A) > 0，在 A 已发生的条件下 B 发生的条件概率：

条件概率公式 P(B|A) = P(AB) / P(A)

条件概率 P(·|A) 本身也满足概率的三条公理（非负性、规范性、可列可加性）。

计算方法：① 在缩减后的样本空间 A 中计算；② 在原样本空间中用定义计算。

定理乘法公式

乘法公式 P(AB) = P(A) · P(B|A) = P(B) · P(A|B)
P(ABC) = P(A) · P(B|A) · P(C|AB)
2.2 事件的独立性

定义独立性

  • 两事件独立：P(AB) = P(A)·P(B)
  • 两两独立：任意两个事件都相互独立（共 C(n,2) 个条件）
  • 相互独立：任意有限个事件的交的概率 = 各自概率之积（共 2ⁿ - n - 1 个条件）
🔑 关键辨析：
  • 互斥：AB = ∅（不能同时发生）
  • 独立：P(AB) = P(A)P(B)（一个发生不影响另一个的概率）
  • 若 P(A) > 0 且 P(B) > 0，则互斥一定不独立，独立一定不互斥
  • 两两独立 ⇏ 相互独立（需验证所有子集的交）

独立性性质：

  • 若 A, B 独立，则 A 与 B̄、Ā 与 B、Ā 与 B̄ 也独立
  • 概率为 0 或 1 的事件与任何事件独立
  • 小概率事件：单个概率虽小，大量独立试验中至少发生一次的概率可能很大（例：1500 人中有人带感冒病毒的概率 ≈ 0.95）
2.3 全概率公式与贝叶斯公式

定义样本空间的划分

设 B₁, B₂, …, Bₙ 互斥且 ∪Bᵢ = Ω，称 {Bᵢ} 为完备事件组（划分）。

定理全概率公式

全概率公式 P(A) = Σᵢ P(Bᵢ) · P(A|Bᵢ)

意义：将复杂事件分解为若干简单事件，"化整为零，各个击破"。

定理贝叶斯公式

贝叶斯公式 P(Bⱼ|A) = P(Bⱼ) · P(A|Bⱼ) / Σᵢ P(Bᵢ) · P(A|Bᵢ)

其中 P(Bⱼ) 为先验概率（由以往数据得到），P(Bⱼ|A) 为后验概率（得到信息后修正）。

2.4 试验的独立性

定义独立试验与常见分布

  • Bernoulli 试验：只有两个结果（成功/失败）的单次试验
  • n 重 Bernoulli 试验：n 次独立重复的 Bernoulli 试验
二项分布 B(n, p) P(X = k) = C(n,k) · p^k · (1-p)^(n-k)，k = 0, 1, …, n
  • 几何分布 G(p)：首次成功在第 k 次 → P(X=k) = (1-p)^(k-1) · p
  • Pascal 分布（负二项分布）：第 r 次成功在第 k 次 → P(X=k) = C(k-1, r-1) · p^r · (1-p)^(k-r)

随机变量及其分布函数
Random Variables and Distribution Functions
▼
3.1 随机变量

定义随机变量

定义在样本空间 Ω 上的实值函数 X = X(ω)，将随机试验的结果映射为实数。

  • 离散型：所有取值可一一列举
  • 连续型：取值充满某个区间
  • 其他：既非离散型又非连续型
意义：随机变量概念的产生是概率论发展史上的重大事件。引入随机变量后，对随机现象的研究由事件及事件概率扩大为随机变量及其取值规律。
3.2 离散型随机变量与分布律

定义分布律

P(X = xₖ) = pₖ，满足 pₖ ≥ 0 且 Σpₖ = 1。可用表格表示。

分布记号分布律 P(X=k)E(X)D(X)
0-1 分布B(1, p)p^k · (1-p)^(1-k)，k=0,1pp(1-p)
二项分布B(n, p)C(n,k) · p^k · (1-p)^(n-k)npnp(1-p)
Poisson 分布P(λ)e^(-λ) · λ^k / k!λλ
几何分布G(p)(1-p)^(k-1) · p1/p(1-p)/p²

定理Poisson 定理

当 n 大、p 小、np = λ 适中时，二项分布可用 Poisson 分布近似：

Poisson 近似 C(n,k) · p^k · (1-p)^(n-k) ≈ e^(-λ) · λ^k / k!
应用场景：电话呼唤次数、交通事故次数、放射性粒子计数等均服从 Poisson 分布。
3.3 连续型随机变量

定义概率密度函数 (PDF)

若存在 f(x) ≥ 0 使 F(x) = ∫₋∞ˣ f(t)dt，则 f(x) 为概率密度函数。

性质：① f(x) ≥ 0；② ∫₋∞^+∞ f(x)dx = 1；③ P(a < X ≤ b) = ∫ₐᵇ f(x)dx

⚠️ 注意：连续型随机变量取任一指定值的概率为零，即 P(X = a) = 0。因此区间的开闭不影响概率。
分布记号概率密度 f(x)E(X)D(X)
均匀分布U(a, b)1/(b-a)，a < x < b(a+b)/2(b-a)²/12
指数分布Exp(λ)λe^(-λx)，x > 01/λ1/λ²
标准正态N(0, 1)(1/√2π) · e^(-x²/2)01
一般正态N(μ, σ²)(1/σ√2π) · e^(-(x-μ)²/(2σ²))μσ²

证明指数分布的无记忆性

无记忆性 P(X > s + t | X > s) = P(X > t)，s, t > 0

证：P(X > s+t | X > s) = P(X > s+t) / P(X > s) = e^(-λ(s+t)) / e^(-λs) = e^(-λt) = P(X > t) ■

性质正态分布

  • 标准化：若 X ~ N(μ, σ²)，则 Z = (X - μ) / σ ~ N(0, 1)
  • P(a < X < b) = Φ((b-μ)/σ) - Φ((a-μ)/σ)
  • Φ(-x) = 1 - Φ(x)
  • 正态分布是最重要的分布：测量误差、身高体重等近似服从；二项分布、Poisson 分布的极限分布是正态分布
3.4 分布函数

定义分布函数 F(x)

分布函数定义 F(x) = P(X ≤ x)，-∞ < x < +∞

性质：

  • 单调不减：x₁ < x₂ ⟹ F(x₁) ≤ F(x₂)
  • 右连续：F(x⁺) = F(x)
  • 边界条件：F(-∞) = 0，F(+∞) = 1
  • P(a < X ≤ b) = F(b) - F(a)
离散型：F(x) = Σ_{xₖ ≤ x} pₖ（阶梯函数）
连续型：F(x) = ∫₋∞ˣ f(t)dt，且 F'(x) = f(x)（在连续点处）

题型：① 离散型分布函数 ② 连续型分布函数 ③ 离散型随机变量函数的分布函数 ④ 连续型随机变量函数的分布函数
3.5 附录：泊松过程

定义泊松过程

满足平稳性（概率规律不随时间推移改变）和独立增量性的计数过程。在长度为 t 的时间区间内，事件发生次数服从参数为 λt 的 Poisson 分布。

随机向量及其分布
Random Vectors and Their Distributions
▼
4.1 随机向量及其联合分布

定义联合分布函数

联合分布函数 F(x, y) = P(X ≤ x, Y ≤ y)

性质：

  • 对 x, y 分别单调不减、右连续
  • F(-∞, y) = F(x, -∞) = 0，F(+∞, +∞) = 1
  • P(a < X ≤ b, c < Y ≤ d) = F(b,d) - F(a,d) - F(b,c) + F(a,c) ≥ 0

定义边缘分布函数

边缘分布函数 Fₓ(x) = F(x, +∞) = P(X ≤ x)
Fᵧ(y) = F(+∞, y) = P(Y ≤ y)
⚠️ 关键：联合分布唯一确定边缘分布，但边缘分布不能唯一确定联合分布！联合分布还包含 X 与 Y 之间关系的信息。
4.2 离散型随机向量

定义联合分布律与边缘分布律

pᵢⱼ = P(X = xᵢ, Y = yⱼ)，满足 pᵢⱼ ≥ 0，Σᵢ Σⱼ pᵢⱼ = 1

边缘分布律 P(X = xᵢ) = pᵢ· = Σⱼ pᵢⱼ
P(Y = yⱼ) = p·ⱼ = Σᵢ pᵢⱼ

定理独立性判别（离散型）

X, Y 独立 ⟺ 对所有 i, j：pᵢⱼ = pᵢ· · p·ⱼ

例：有放回摸球 → 独立；不放回摸球 → 不独立。两者边缘分布相同但联合分布不同。
4.3 连续型随机向量

定义联合密度与边缘密度

F(x,y) = ∫₋∞ˣ ∫₋∞ʸ f(u,v) dv du（Fubini 定理保证可交换积分次序）

边缘概率密度 fₓ(x) = ∫₋∞^+∞ f(x, y) dy
fᵧ(y) = ∫₋∞^+∞ f(x, y) dx

定理独立性判别（连续型）

X, Y 独立 ⟺ f(x,y) = fₓ(x) · fᵧ(y)（几乎处处成立）

⚠️ 注意：若联合密度的支撑区域不是矩形区域，则 X, Y 一定不独立。二维正态分布中若 ρ ≠ 0，则 X, Y 不独立。
4.4 随机向量的函数分布

定理和的分布（卷积公式）

Z = X + Y 的密度 f_Z(z) = ∫₋∞^+∞ f(x, z-x) dx = ∫₋∞^+∞ f(z-y, y) dy

独立时：f_Z(z) = ∫₋∞^+∞ fₓ(x) · fᵧ(z-x) dx

定理商的分布

Z = X/Y 的密度 f_Z(z) = ∫₋∞^+∞ |y| · f(zy, y) dy

定理最大值与最小值的分布

设 X₁, …, Xₙ 独立同分布，分布函数为 F(x)：

极值分布 M = max(X₁,…,Xₙ)：Fₘ(x) = [F(x)]^n
N = min(X₁,…,Xₙ)：Fₙ(x) = 1 - [1 - F(x)]^n
4.5 条件分布

定义条件分布

离散型：P(X=xᵢ|Y=yⱼ) = pᵢⱼ / p·ⱼ

条件密度（连续型） f_{X|Y}(x|y) = f(x, y) / fᵧ(y)，fᵧ(y) > 0
f_{Y|X}(y|x) = f(x, y) / fₓ(x)，fₓ(x) > 0

随机变量的数字特征
Numerical Characteristics of Random Variables

5.1 数学期望

定义数学期望

离散型 E(X) = Σₖ xₖ · pₖ（要求级数绝对收敛）
连续型 E(X) = ∫₋∞^+∞ x · f(x) dx（要求积分绝对收敛）

说明：E(X) 是一个实数（加权平均），不是变量。绝对收敛保证级数和不随排列次序改变。

定理随机变量函数的期望

一维 E[g(X)] = Σₖ g(xₖ) · pₖ（离散）
E[g(X)] = ∫ g(x) · f(x) dx（连续）
二维 E[g(X, Y)] = ∫∫ g(x,y) · f(x,y) dx dy（连续）
E[g(X, Y)] = Σᵢ Σⱼ g(xᵢ, yⱼ) · pᵢⱼ（离散）
优势：不必先求 g(X) 的分布，可直接用 X 的分布计算。

性质期望的性质

  • E(C) = C
  • E(aX + b) = aE(X) + b
  • E(X + Y) = E(X) + E(Y)
  • 若 X, Y 独立，则 E(XY) = E(X) · E(Y)
  • 推广：E(ΣaᵢXᵢ) = ΣaᵢE(Xᵢ)
  • E(X²) ≥ [E(X)]²（Jensen 不等式，等号当且仅当 X 为常数）

应用数学期望的实际意义

  • 分赌本问题：按期望分配赌金
  • 核酸检测：分组检测最优方案（k=10 或 11 时节省约 80% 工作量）
  • 投资决策：比较不同方案的期望收益
5.2 方差

定义方差与标准差

方差公式 D(X) = E[X - E(X)]² = E(X²) - [E(X)]²
σ(X) = √D(X)（标准差）

性质方差的性质

  • D(C) = 0
  • D(aX + b) = a²D(X)
  • 若 X, Y 独立，则 D(X ± Y) = D(X) + D(Y)
  • 推广：若 X₁, …, Xₙ 独立，则 D(ΣaᵢXᵢ) = Σaᵢ²D(Xᵢ)
分布E(X)D(X)
B(1, p)pp(1-p)
B(n, p)npnp(1-p)
P(λ)λλ
G(p)1/p(1-p)/p²
U(a, b)(a+b)/2(b-a)²/12
Exp(λ)1/λ1/λ²
N(μ, σ²)μσ²

定理Chebyshev 不等式

Chebyshev 不等式 P(|X - E(X)| ≥ ε) ≤ D(X) / ε²
等价形式：P(|X - μ| < ε) ≥ 1 - σ²/ε²

意义：即使不知道分布，也能给出概率的上界估计。但估计较粗糙，若已知分布则直接计算更精确。

5.3 协方差与相关系数

定义协方差

协方差 Cov(X, Y) = E[(X - EX)(Y - EY)] = E(XY) - E(X)·E(Y)

性质：

  • Cov(X, X) = D(X)
  • Cov(X, Y) = Cov(Y, X)
  • Cov(aX + b, cY + d) = ac · Cov(X, Y)
  • D(X ± Y) = D(X) + D(Y) ± 2Cov(X, Y)
  • Cov(X, Y) = 0 ⟺ X, Y 不相关

定义相关系数

相关系数 ρ(X,Y) = Cov(X,Y) / √(D(X)·D(Y))

性质（Cauchy-Schwarz 不等式）：

  • |ρ| ≤ 1
  • |ρ| = 1 ⟺ Y = aX + b（几乎处处线性相关）
  • ρ = 0 称为不相关

辨析不相关 vs 独立

🔑 核心区别：
  • 独立 → 不相关（Cov = 0，ρ = 0）
  • 不相关 ⇏ 独立（只说明无线性关系，可能有非线性关系）
  • 特例：对二维正态分布，不相关 ⟺ 独立

定义协方差矩阵

C = [Cov(Xᵢ, Xⱼ)] 为 n×n 矩阵，性质：非负定（半正定）。

6.1 一维随机变量的特征函数

定义特征函数

将 Fourier 变换应用于分布函数，引入特征函数。它既能完全决定分布函数，又具有良好的分析性质。

特征函数定义 φₓ(t) = E(e^(itX)) = E(cos tX) + i·E(sin tX)

离散型：φ(t) = Σₖ e^(itxₖ) · pₖ
连续型：φ(t) = ∫₋∞^+∞ e^(itx) · f(x) dx

特征函数总是存在（因为 |e^(itx)| = 1，积分/求和总是收敛的）。

性质特征函数的性质

  • 性质 1：φ(0) = 1，|φ(t)| ≤ 1
  • 性质 2：φ(t) 在 (-∞, +∞) 上一致连续
  • 性质 3：非负定性
  • 性质 4：若 X, Y 独立，则 φ_{X+Y}(t) = φₓ(t) · φᵧ(t)（独立和的特征函数 = 特征函数之积）
  • 推论（再生性）：同类型分布之和仍为该类型（Poisson、正态、Gamma 等）
  • 性质 5：若 Y = aX + b，则 φᵧ(t) = e^(ibt) · φₓ(at)
  • 性质 6：若 E|X|ⁿ 存在，则 φ⁽ⁿ⁾(0) = iⁿ · E(Xⁿ)，即 E(Xⁿ) = i⁻ⁿ · φ⁽ⁿ⁾(0)

定理常见分布的特征函数

分布特征函数 φ(t)
B(n, p)(pe^(it) + q)^n
P(λ)exp{λ(e^(it) - 1)}
U(a, b)(e^(itb) - e^(ita)) / (it(b-a))
Exp(λ)(1 - it/λ)^(-1)
N(0, 1)e^(-t²/2)
N(μ, σ²)exp{iμt - σ²t²/2}

定理反演公式与唯一性定理

由特征函数可以唯一确定分布函数（Levy 反演公式）。

唯一性定理：随机变量的分布函数由其特征函数唯一决定。这是特征函数作为刻画随机变量统计规律数学工具的理论基础。
6.2 多维随机变量的特征函数

定义二维特征函数

二维特征函数 φ(t₁, t₂) = E[exp(i(t₁X + t₂Y))]

连续型：φ(t₁, t₂) = ∫∫ e^(it₁x + it₂y) · f(x,y) dx dy

性质：

  • φ(0, 0) = 1
  • X, Y 独立 ⟺ φ(t₁, t₂) = φₓ(t₁) · φᵧ(t₂)
  • 边缘特征函数：φₓ(t₁) = φ(t₁, 0)，φᵧ(t₂) = φ(0, t₂)
  • 逆转公式和唯一性定理同样成立

7.1 收敛性概念与大数定律

定义四种收敛方式

  • 依分布收敛（弱收敛）：Xₙ →ᵈ X，即 Fₙ(x) → F(x)（在 F 的连续点处）
  • 依概率收敛：Xₙ →ᵖ X，即 ∀ε > 0，P(|Xₙ - X| > ε) → 0
  • r 阶收敛：Xₙ →ʳ X，即 E|Xₙ - X|ʳ → 0
  • 以概率 1 收敛（a.s. 收敛）：Xₙ →ᵃ.ˢ. X，即 P(lim Xₙ = X) = 1
收敛强度关系：
a.s. 收敛 ⟹ 依概率收敛 ⟹ 依分布收敛
r 阶收敛 ⟹ 依概率收敛
反向均不成立（一般情况下）

定义大数定律

设 {Xₙ} 为随机变量序列，E(Xᵢ) 存在，令 Sₙ = ΣXᵢ，若 Sₙ/n - E(Sₙ/n) →ᵖ 0，则称 {Xₙ} 服从大数定律。

定理Bernoulli 大数定律

nₐ 为 n 次 Bernoulli 试验中事件 A 发生的次数，p = P(A)：

Bernoulli 大数定律 ∀ε > 0，lim P(|nₐ/n - p| ≥ ε) = 0

意义：当 n 很大时，频率与概率有较大偏差的可能性很小。可用频率代替概率。

定理Poisson 大数定律

设各次试验中 A 发生的概率不同，第 i 次为 pᵢ，则：

Poisson 大数定律 (1/n)Σ Xᵢ - (1/n)Σ pᵢ →ᵖ 0

定理Chebyshev 大数定律

设 X₁, X₂, … 相互独立，E(Xᵢ) = μᵢ，D(Xᵢ) ≤ C，则：

Chebyshev 大数定律 (1/n)ΣXᵢ - (1/n)Σμᵢ →ᵖ 0

定理Khintchine 大数定律（最重要）

设 X₁, X₂, … 独立同分布 (i.i.d.)，E(Xᵢ) = μ，则：

Khintchine 大数定律 X̄ₙ = (1/n)ΣXᵢ →ᵖ μ
核心意义：样本均值依概率收敛于总体均值。为"用频率估计概率"和"用样本均值估计期望"提供了理论依据。
7.2 中心极限定理

定义中心极限定理

研究独立随机变量之和的极限分布为正态分布的条件。

定理De Moivre-Laplace 定理

设 Sₙ ~ B(n, p)：

De Moivre-Laplace (Sₙ - np) / √(np(1-p)) →ᵈ N(0, 1)

意义：正态分布是二项分布的极限分布，n 充分大时可用正态分布近似二项分布。

⚠️ 注意：泊松分布和正态分布都是二项分布的极限分布。当 n 大 p 小（np 适中）时，泊松近似比正态近似更精确。正态近似只需 n 大。

定理独立同分布中心极限定理（Lindeberg-Lévy）

设 X₁, X₂, … i.i.d.，E(Xᵢ) = μ，D(Xᵢ) = σ² > 0，则：

Lindeberg-Lévy CLT (ΣXᵢ - nμ) / (σ√n) →ᵈ N(0, 1)

意义：这是数理统计中大样本统计推断的理论基础。

定理Lyapunov 定理

设 X₁, X₂, … 相互独立（不必同分布），满足 Lyapunov 条件，则：

Lyapunov CLT (ΣXᵢ - Σμᵢ) / Bₙ →ᵈ N(0, 1)，其中 Bₙ² = Σσᵢ²

定理表明：在相当一般的条件下，独立随机变量个数增加时，其和的分布趋于正态分布。

应用中心极限定理的应用

核心思想：大量独立随机变量之和近似服从正态分布。
近似计算公式 P(a ≤ Sₙ ≤ b) ≈ Φ((b - nμ)/(σ√n)) - Φ((a - nμ)/(σ√n))

典型应用：

  • 保险公司利润计算（万人投保，死亡率 0.006）
  • 电话分机外线配备（260 部分机，4% 使用率，需 16 条外线）
  • 船舶纵摇角概率（90000 次波浪冲击）
  • 家长会人数估计（400 名学生）
