第1章 通信系统导论

1.1 通信系统的组成

通信系统五大组成部分：输入变换装置 → 发送设备 → 传输信道 → 接收设备 → 输出变换装置

1.1.2 无线电发送设备

  • 组成：高频振荡 → 缓冲 → 倍频 → 高频放大 → 调制 → 天线发射
  • 调制的目的：将低频信号"装载"到高频载波上，减小天线尺寸，区分不同电台
  • 调制方式：调幅(AM)、调频(FM)、调相(PM)

1.1.3 无线电接收设备（超外差式接收机）

超外差接收机原理：fi = fo - fs
组成：高频放大 → 混频 → 中频放大 → 检波 → 低频放大
优点：灵敏度高、选择性好、工作稳定、波段工作质量指标一致性好

1.2 信号的频谱表示法

  • 任何复杂信号可分解为不同频率正弦信号之和
  • 带宽：信号最高频率与最低频率之差

1.3 无线电波段划分

波段频率范围频段名称
长波30~300kHz低频(LF)
中波0.3~1.5MHz中频(MF)
短波1.5~30MHz高频(HF)
超短波30~300MHz甚高频(VHF)
分米波0.3~3GHz特高频(UHF)
厘米波3~30GHz超高频(SHF)

1.3.2 无线电波传播方式

  • 地波：沿地面传播（长波、超长波）
  • 空间波：直线传播（超短波）
  • 天波：电离层反射（短波，300kHz~30MHz）
第2章 通信电子线路分析基础

2.0 元件的高频特性

高频特性总结：
• 电阻：高频时 = 电阻 + 分布电容(CR) + 引线电感(LR)
• 电感：高频时 = 电感 + 损耗电阻(r) + 分布电容
• 电容：高频时 = 电容 + 损耗电阻 + 分布电感
• 集肤效应：频率越高，电流越向导线表面集中

2.1 选频网络

2.1.1 串联谐振回路

回路阻抗
Z = R + jX = R + j(ωL - 1/ωC)
谐振频率
ω₀ = 1/√(LC)，f₀ = 1/(2π√(LC))
特性阻抗
ρ = ω₀L = 1/(ω₀C) = √(L/C)
品质因数
Q = ω₀L/R = 1/(ω₀CR) = ρ/R = (1/R)√(L/C)
谐振时电压关系（电压谐振）
VL0 = VC0 = QVs
广义失谐系数
ξ = X/R = (ωL - 1/ωC)/R ≈ Q·2Δω/ω₀
谐振曲线（归一化）
N(f) = I/I₀ = 1/√(1 + ξ²) = 1/√[1 + (Q·2Δf/f₀)²]
通频带
B = 2Δf₀.₇ = f₀/Q
矩形系数（单回路）
Kr0.1 = BW₀.₁/BW₀.₇ = √(100-1) = 9.95
有载Q值
QL = ω₀L/(R + RS + RL)

2.1.2 并联谐振回路

谐振频率
ωp = 1/√(LC)，fp = 1/(2π√(LC))
谐振电阻
Rp = L/(RC) = Q²R = (ω₀L)²/R
品质因数
Qp = Rp/(ωpL) = ωpC·Rp = Rp/ρ
谐振时电流关系（电流谐振）
ICp = ILp = Qp·IS
有载Q值
QL = Q0 / (1 + Rp/RS + Rp/RL)
串并联谐振回路对比要点：
• 串联：阻抗最小(R)，电压谐振，f>f₀呈感性，f • 并联：阻抗最大(Rp)，电流谐振，f>f₀呈容性，f • 通频带：B = f₀/Q（两者相同）

2.1.3 串并联阻抗等效互换与抽头变换

串→并联等效（Q>>1时）
X并 = X串，R并 = Q²R串
抽头阻抗变换
接入系数 p = VT/V（p<1）
R'S = RS/p²（低抽头→高抽头，电阻提高1/p²倍）
C'L = p²CL
插入损耗
Kl(dB) = 20lg[1/(1 - QL/Q0)]

2.1.4 耦合回路

耦合系数
电感耦合：k = M/√(L₁L₂)，若L₁=L₂=L，则k=M/L
电容耦合：k = CM/(C+CM)
反射阻抗
Zf1 = (ωM)²/Z22（次级反射到初级）
Zf2 = (ωM)²/Z11（初级反射到次级）
反射电阻永远为正，反射电抗与原回路电抗性质相反
耦合因数
η = k/kc = kQ
η<1：弱耦合；η=1：临界耦合；η>1：强耦合
临界耦合系数
kc = 1/Q（当Q₁=Q₂=Q时）
耦合回路频率特性
α = I₂/I₂max = 2η / √[(1+η²-ξ²)² + 4ξ²]
临界耦合时通频带
2Δf₀.₇ = √2 · f₀/Q（比单回路宽√2倍）
临界耦合时矩形系数
Kr0.1 = √(100^(1/2)-1)·f₀/Q ≈ 3.16（远优于单回路的9.95）

2.2 非线性电路分析基础

2.2.2 非线性电路的分析方法

幂级数分析法
i = a₀ + a₁v + a₂v² + a₃v³ + ...

当v = V₁cosω₁t + V₂cosω₂t时，产生的频率分量：
直流、ω₁、ω₂、2ω₁、2ω₂、ω₁±ω₂、3ω₁、3ω₂、2ω₁±ω₂、ω₁±2ω₂ ...

结论：最高谐波次数不超过n；组合频率pω₁±qω₂满足p+q≤n
折线分析法（大信号）
ic = 0（vBE < VBZ）
ic = gc(vBE - VBZ)（vBE > VBZ）
gc为跨导，VBZ为截止电压
线性时变参量分析法
i(t) ≈ I₀(t) + g(t)·v₂(t)
g(t) = g₀ + g₁cosω₁t + g₂cos2ω₁t + ...
频率变换：gncos(nω₁t)·V₂cosω₂t → 产生(nω₁±ω₂)分量

2.1.5 选择性滤波器

  • LC集中滤波器：带通滤波器
  • 石英晶体滤波器：Q值极高（几万~几百万），C₀>>Cq
  • 陶瓷滤波器：Q值几百
  • 声表面波滤波器(SAWF)：工作频率10MHz~1GHz，矩形系数可达1.2
第3章 高频小信号放大器

3.1 概述

质量指标：
① 增益：AV=Vo/Vi，Ap=Po/Pi
② 通频带：B=2Δf₀.₇（增益下降到0.707倍时的频率范围）
③ 选择性：矩形系数Kr、抑制比dn
④ 工作稳定性
⑤ 噪声系数：NF = (Psi/Pni)/(Pso/Pno)

3.2 晶体管高频小信号等效电路

3.2.1 Y参数等效电路

Y参数方程
İ₁ = yieV̇₁ + yreV̇₂
İ₂ = yfeV̇₁ + yoeV̇₂

3.2.2 混合π等效电路

主要参数：
rbb'：基极电阻 | rb'e：发射结电阻 | Cb'e：发射结电容
Cb'c：集电结电容 | gm：跨导 = Ic/26mV | gmv̇b'e：等效电流源
Y参数与混合π参数的转换
yie = Yb'e/(1 + rbb'Yb'e)，Yb'e = jω(Cb'e+Cb'c)
yfe = gm/(1 + rbb'Yb'e)
yre = -jωCb'c/(1 + rbb'Yb'e)
yoe = jωCb'c(1 + gmrbb'/(1+rbb'Yb'e))

3.2.4 晶体管高频参数

截止频率
β = β₀/√(1+(f/fβ)²)
fβ：|β|下降到β₀/√2时的频率
特征频率
fT = β₀·fβ ≈ fβ·√(β₀²-1)
|β|=1时的频率，β·f ≈ fT
最高振荡频率
fmax = (1/2π)√(gm/(4rbb'Cb'eCb'c))
频率关系：fmax > fT > fβ

3.3 晶体管谐振放大器

3.3.1 单级单调谐回路谐振放大器

谐振电压增益
AV0 = -p₁p₂yfe / gΣ
gΣ = p₁²goe + gp + p₂²gie2
CΣ = p₁²Coe + C + p₂²Cie2
通频带
2Δf₀.₇ = f₀/QL
带宽增益积为常数：AV0·2Δf₀.₇ = |yfe|/(2πCΣ)
矩形系数
Kr0.1 = √(10²-1) ≈ 9.95（与单回路相同，选择性差）

3.3.2 多级单调谐放大器

m级级联
总增益：Am = AV1^m
通频带：(2Δf₀.₇)m = √(2^(1/m)-1) · (2Δf₀.₇)单级
带宽缩减因子：X = √(2^(1/m)-1)

3.4 谐振放大器的稳定性

自激条件
(YS+Yi)(yoe+Y'L) = yfe·yre
稳定系数
S = (YS+Yi)(yoe+Y'L)/(yfe·yre)
S=1自激；S<1更自激；S>>1稳定（工程取S=5~10）
稳定电压增益
(AV0)S = |yfe|/(2.5ω₀Cre)（当S=5时）
克服自激的方法：
① 中和法：外加反馈抵消yre的内反馈，CN≈(N₁/N₂)Cb'c
② 失配法：使YL或YS增大，输出与输入不匹配，常用共发-共基级联

3.5 放大电路的噪声

电阻热噪声
U²n = 4kTRB，I²n = 4kTGB
k = 1.38×10⁻²³ J/K（波尔兹曼常数）
噪声系数
NF = (Psi/Pni)/(Pso/Pno) = 输入信噪比/输出信噪比
NF(dB) = 10lgNF
级联噪声系数
NF = NF1 + (NF2-1)/GP1 + (NF3-1)/(GP1GP2) + ...
结论：前级影响最大，应降低前级噪声系数并增大前级功率增益
第4章 谐振功率放大器

4.1 概述

工作状态分类：
甲类：θc=180°，η=50% | 乙类：θc=90°，η=78.5% | 丙类：θc<90°，η>78.5%

4.2 工作原理

高频功放特点：
① 放大高频大信号，非线性工作状态
② 基极负偏压，丙类工作（θc<90°）
③ 集电极电流为尖顶余弦脉冲
④ 负载为LC谐振回路，输出正弦波
外部电路关系
vBE = -VBB + Vbmcosωt
vCE = VCC - Vcmcosωt
ic = gc(vBE - VBZ)

4.2.2 功率关系与效率

功率与效率
直流功率：P= = VCC·Ic0
输出功率：Po = (1/2)Vcm·Icm1 = (1/2)I²cm1Rp
集电极效率：ηc = Po/P= = (1/2)ξ·g₁(θc)

电压利用系数：ξ = Vcm/VCC
波形系数：g₁(θc) = Icm1/Ic0

4.3 折线近似分析法

导通角
cosθc = (VBB + VBZ)/Vbm
集电极电流脉冲
Icmax = gcVbm(1 - cosθc)
ic = Icmax·(cosωt - cosθc)/(1 - cosθc)
傅里叶分解
Ic0 = Icmax·α₀(θc)
Icm1 = Icmax·α₁(θc)
Icmn = Icmax·αn(θc)

α₀(θc) = (sinθc - θccosθc)/[π(1-cosθc)]
α₁(θc) = (θc - cosθcsinθc)/[π(1-cosθc)]
最佳通角
θc ≈ 70°（兼顾功率与效率）
θc=120°时α₁最大（输出功率最大但效率低）
θc→0时η→100%（但功率→0）

4.3.4 动态特性与负载特性

动态特性方程
ic = gd(vCE - V₀)
gd = -(Vbm/Vcm)·gc（斜率为负，相当于负电阻）
三种工作状态：
• 欠压状态：输出功率和效率较低，输出电压随负载变化
• 临界状态：输出功率最大，效率较高，发射机末级常用
• 过压状态：输出电压平稳，发射机中间级和集电极调幅常用
最佳负载电阻
Rp = (ξVCC)²/(2Po)
临界时ξ≈1：Rp ≈ V²CC/(2Po)

4.4 谐振功率放大器电路

馈电电路：
• 串馈：直流电源、负载回路、功率管三者串联
• 并馈：三者并联，调谐电容上无高压，安全可靠
• 自偏压：Ib0或Ie0在电阻上产生负偏压

4.6 丙类倍频器

  • 输出回路调谐于n次谐波频率
  • 二次谐波最佳通角≈60°，三次谐波≈40°
  • 输出回路Q值要求：Qo > 10nπ
第5章 正弦波振荡器

5.2 反馈型振荡器的工作原理

5.2.2 起振条件

起振条件
ÂF̂ > 1
即：{ A₀F > 1（振幅条件）
　　 { φA + φF = 2nπ（相位条件，正反馈）

5.2.3 平衡条件

平衡条件
Â·F̂ = 1
即：{ AF = 1（振幅平衡）
　　 { φA + φF = 2nπ（相位平衡）

5.2.4 稳定条件

振幅稳定条件
∂A/∂Vom |Vom=VomQ < 0
靠有源器件工作在非线性区实现
相位（频率）稳定条件
∂φZ/∂ω < 0
靠并联谐振回路的相频特性实现

5.3 反馈型LC振荡器线路

5.3.2 三端式LC振荡器

三端式振荡器组成法则（相位判断准则）
Xbe与Xce电抗性质相同
Xcb与Xbe、Xce电抗性质相反
简记："ce同抗件，cb反抗件"
电路类型名称反馈系数F振荡频率特点
电感三端式哈特莱(Hartley)(L₂+M)/(L₁+M)≈1/(2π√(LC))易起振，波形差，频率不宜太高
电容三端式考毕兹(Colpitts)C₁/C₂≈1/(2π√(LC'))，C'=C₁C₂/(C₁+C₂)波形好，频率稳定度较高
串联改进克拉泼(Clapp)C₁/(C₁+C₂)≈1/(2π√(LC₃))频率稳定度高，不适合波段振荡
并联改进西勒(Seiler)C₁/(C₁+C₂)≈1/(2π√(L(C₃+C₄)))波段覆盖宽，适合波段振荡

5.4 振荡器的频率稳定

频率稳定度
相对频率稳定度：δ = Δfmax/f₀ |t=Δt
相对频率变化：Δω/ω₀ ≈ -(1/2)(ΔL/L + ΔC/C)
提高频率稳定度的措施：
① 减小外因变化（恒温、稳压、密封、屏蔽）
② 提高回路标准性（优质LC材料）
③ 减小φYF及其变化量（f<T，选电容三端式）
④ 减小负载影响（加缓冲级，松耦合）

5.5 石英晶体振荡器

石英晶体等效参数
Lq：10³~10²H | Cq：10⁻⁴~10⁻¹pF | rq：几十~几百Ω | C₀：1~10pF
Q值：几万~几百万
串联谐振频率与并联谐振频率
fq = 1/(2π√(LqCq))
fp = fq√(1 + Cq/C₀) ≈ fq(1 + Cq/(2C₀))
fq < fp，两者间距很小
晶体振荡器电路：
• 皮尔斯(Pierce)电路：晶振等效为电感，振荡频率f₀≈fq(1+Cq/(C₀+CL))
• 密勒(Miller)电路：晶振接栅极和源极之间，用场效应管
• 串联型：晶振工作在串联谐振频率fq，等效为短路
第6章 调幅、检波与混频（频谱搬移电路）

6.2 振幅调制原理

6.2.2 调幅波的性质

普通调幅波(AM)
v(t) = V₀(1 + macosΩt)cosω₀t
展开：= V₀cosω₀t + (maV₀/2)cos(ω₀+Ω)t + (maV₀/2)cos(ω₀-Ω)t

调幅指数：ma = KdVΩ/V₀（要求ma≤1）
调幅度测量
ma = (Vmax - Vmin)/(Vmax + Vmin) = (Vmax - V₀)/V₀
频带宽度
BAM = 2Fmax（调制信号最高频率的两倍）
功率关系
载波功率：PoT = V₀²/(2R)
每个边频功率：PSB = (ma²/4)PoT
总功率：PAM = (1 + ma²/2)PoT
结论：ma≤1时，边频功率最多占总功率的1/3，效率低

6.2.3 DSB与SSB调幅

双边带调幅(DSB-SC)
vDSB(t) = V₀cosΩt·cosω₀t
频带宽度：BDSB = 2Fmax
单边带调幅(SSB-SC)
vSSB(t) = (maV₀/2)cos(ω₀±Ω)t
频带宽度：BSSB = Fmax

6.3 振幅调制电路

调幅电路分类：
低电平调幅：平方律调幅、平衡调幅器、环形调制器、模拟相乘器
高电平调幅：集电极调幅（过压区）、基极调幅（欠压区）
模拟相乘器调幅
vo(t) = kV₀cosω₀t·(VDC+VΩcosΩt)
调节VDC可改变调幅系数ma

6.4 振幅解调（检波）

6.4.2 二极管峰值包络检波器

电压传输系数
Kd = cosθ ≈ 1（R>>Rd时）
θ ≈ ∛(3πRd/(R))
等效输入电阻
Rid ≈ R/2（Kd≈1时）
惰性失真条件（不产生失真）
Ωmax·RC ≤ 1.5/√(1-ma²)
或简记：Ωmax·RC ≤ 1.5
负峰切割失真条件
ma < Rg/(R+Rg) = R//Rg/R

6.4.3 同步检波

适用：DSB、SSB信号解调（包络检波只能解调AM信号）
原理：本地载波与信号相乘 → 低通滤波 → 恢复调制信号
vΩ = (1/2)V₁V₀cosφcosΩt（φ为相位差，φ=0时输出最大）

6.5 混频器

混频原理
fi = fo - fs（低中频）或 fi = fo + fs（高中频）
中频信号保持原调制规律不变
变跨导混频
gc = g₁/2（变频跨导）
Ii = gc·Vsm/2
混频电压增益：Avc = gc/(goc+gL)

6.5.5 混频器的干扰

组合频率干扰
fk = |±pfo ± qfs|（p,q为正整数）
干扰哨叫条件：fs = (p±1)/(q-p)·fi
镜像频率干扰
f's = fs + 2fi = fo + fi
抑制方法：提高前端选择性、提高中频频率

7.2 调角波的性质

7.2.1 FM波和PM波的数学表达式

调频波(FM)
瞬时频率：ω(t) = ω₀ + kfvΩ(t)
瞬时相位：θ(t) = ω₀t + kf∫vΩ(t)dt
af(t) = Vmcos[ω₀t + kf∫vΩ(t)dt]
调相波(PM)
瞬时相位：θ(t) = ω₀t + kpvΩ(t)
瞬时频率：ω(t) = dθ/dt = ω₀ + kpdvΩ/dt
ap(t) = Vmcos[ω₀t + kpvΩ(t)]
单音调制时（vΩ=VΩcosΩt）
FM波：af(t) = Vmcos(ω₀t + mfsinΩt)
频偏：Δωm = kfVΩ（与Ω无关）
调制指数：mf = kfVΩ/Ω（与Ω成反比）

PM波：ap(t) = Vmcos(ω₀t + mpcosΩt)
频偏：Δωm = kpVΩΩ（与Ω成正比）
调制指数：mp = kpVΩ（与Ω无关）
通用关系
Δωm = m·Ω 或 Δf = m·F
注意：角度调制中m可大于1（不同于AM的ma≤1）

7.2.2 调角信号的频谱

FM波的贝塞尔函数展开
af(t) = V₀∑Jn(mf)cos(ω₀+nΩt) （n从-∞到+∞）

频谱特点：
• 载频和无穷多对上下边频组成
• 各频率分量间距为Ω
• 振幅由贝塞尔函数Jn(mf)决定
• 奇数次上下边频相位相反
有效频带宽度
BW = 2(m+1)F = 2(Δf + F)
结论：FM为恒带宽调制（mf↑则带宽基本不变）
PM的带宽随F变化（mp不变，F↑则带宽↑）
功率
Pav = V₀²/(2RL)
结论：调频波平均功率等于未调载波功率，调制只是重新分配功率

7.3 调频方法及电路

调频方法：
直接调频：用调制信号控制振荡器频率（变容二极管、电抗管）
优点：频偏大 | 缺点：中心频率不稳定

间接调频：先积分再调相（调制信号积分→调相→等效调频）
优点：频率稳定度高 | 缺点：频偏小
变容二极管直接调频
Cj = Cj0/(1+vR/VD)γ

Δω(t)/ω₀ ≈ -(1/2)ΔC/C

当γ=1时，二次、三次非线性失真为零，调制线性最好

7.4 调角信号解调

鉴频方法：
① 斜率鉴频器（利用谐振回路幅频特性的斜率段）
② 相位鉴频器（利用相位-频率特性的线性段）
③ 比例鉴频器（具有自动限幅作用）
④ 锁相环鉴频

7.4.1 鉴频器主要技术指标

  • 鉴频灵敏度：Sd = dvo/df |f=f₀（单位频偏产生的输出电压）
  • 线性鉴频范围：2Δfmax
  • 非线性失真

7.5 调频制的抗干扰性能

FM优于AM的抗干扰性：
• FM波为等幅波，可用限幅器消除幅度干扰
• 调制指数mf越大，抗干扰能力越强（但带宽增大）
• FM的信噪比改善与mf成正比
