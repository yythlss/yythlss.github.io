第四章补充：留数法（PPT内容）

📖 留数法概述

留数法是复变函数论中的重要方法，主要用于计算复变函数的围道积分，特别是在拉普拉斯逆变换中有重要应用。

📐 留数法核心公式

公式1
∮C f(z) dz = 2πj Σ Res[f(z), zk]

留数定理

公式2
Res[f(z), z0] = limz→z₀ (z-z₀)f(z)

一阶极点的留数计算

公式3
Res[f(z), z0] = (1/(n-1)!) limz→z₀ dn-1/dzn-1 [(z-z₀)nf(z)]

n阶极点的留数计算

公式4
f(t) = Σ Res[F(s)est, sk]

拉普拉斯逆变换的留数法

📝 留数法核心概念

概念1

极点：使函数f(z)趋于无穷的点z₀

概念2

留数：函数在极点处的洛朗级数展开中(z-z₀)⁻¹项的系数

概念3

围道积分：沿闭合路径的积分，等于路径内所有极点的留数之和乘以2πj

🎯 留数法应用

计算步骤
  • 确定被积函数的极点
  • 计算每个极点的留数
  • 应用留数定理求积分
常见应用
  • 拉普拉斯逆变换
  • 傅里叶变换计算
  • 复变函数积分
注意事项
  • 正确识别极点阶数
  • 选择合适的积分路径
  • 处理无穷远点

💡 留数法例题

例题：计算 ∮|z|=2 (z²+1)/(z²-1) dz

解：

1. 求极点：z²-1=0，得z=±1（一阶极点）

2. 计算留数：

  Res[f(z), 1] = limz→1 (z-1)(z²+1)/(z²-1) = (1+1)/(1+1) = 1

  Res[f(z), -1] = limz→-1 (z+1)(z²+1)/(z²-1) = (1+1)/(-1-1) = -1

3. 应用留数定理：

  ∮|z|=2 (z²+1)/(z²-1) dz = 2πj(1+(-1)) = 0

🎯 公式速查大全

🔄 傅里叶变换公式

  • F(jω) = ∫-∞∞ f(t)e-jωt dt
  • f(t) = (1/2π) ∫-∞∞ F(jω)ejωt dω
  • ℱ[δ(t)] = 1
  • ℱ[1] = 2πδ(ω)
  • ℱ[ejω₀t] = 2πδ(ω-ω₀)
  • ℱ[cos(ω₀t)] = π[δ(ω-ω₀) + δ(ω+ω₀)]
  • ℱ[sin(ω₀t)] = jπ[δ(ω+ω₀) - δ(ω-ω₀)]
  • ℱ[f(t)*g(t)] = F(jω)G(jω)

📈 拉普拉斯变换公式

  • F(s) = ∫0∞ f(t)e-st dt
  • f(t) = (1/2πj) ∫σ-j∞σ+j∞ F(s)est ds
  • ℒ[δ(t)] = 1
  • ℒ[ε(t)] = 1/s
  • ℒ[eat] = 1/(s-a)
  • ℒ[tn] = n!/sn+1
  • ℒ[cos(ωt)] = s/(s²+ω²)
  • ℒ[sin(ωt)] = ω/(s²+ω²)

📊 Z变换公式

  • F(z) = Σn=-∞∞ f[n]z-n
  • f[n] = (1/2πj) ∮ F(z)zn-1 dz
  • Z[δ[n]] = 1
  • Z[ε[n]] = 1/(1-z-1)
  • Z[anε[n]] = 1/(1-az-1)
  • Z[n anε[n]] = az-1/(1-az-1)²
  • H(z) = Y(z)/F(z)

🔢 卷积公式

  • y(t) = f(t) * h(t) = ∫-∞∞ f(τ)h(t-τ) dτ
  • y[n] = f[n] * h[n] = Σk=-∞∞ f[k]h[n-k]
  • f(t) * δ(t) = f(t)
  • f[n] * δ[n] = f[n]
  • 交换律：f*g = g*f
  • 结合律：f*(g*h) = (f*g)*h
  • 分配律：f*(g+h) = f*g + f*h

📐 留数法公式

  • ∮C f(z) dz = 2πj Σ Res[f(z), zk]
  • Res[f(z), z0] = limz→z₀ (z-z₀)f(z)
  • n阶极点：Res = (1/(n-1)!) lim dn-1/dzn-1 [(z-z₀)nf(z)]
  • 拉普拉斯逆变换：f(t) = Σ Res[F(s)est, sk]

⚙️ 系统分析公式

  • Y(s) = H(s)F(s)
  • Y(z) = H(z)F(z)
  • H(s) = Y(s)/F(s)
  • H(z) = Y(z)/F(z)
  • 状态方程：ẋ = Ax + Bu
  • 输出方程：y = Cx + Du
  • H(s) = C(sI-A)-1B + D
