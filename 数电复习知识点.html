📖 第1章 数字逻辑概述

1.1 数字信号与数字电路

模拟信号与数字信号

模拟信号：时间和幅值上都连续的信号
数字信号：时间和幅值上都离散的信号（常用0、1表示）

数字电路的特点

  • 基本工作信号是二进制数字信号（高电平和低电平）
  • 晶体管处于开关工作状态，抗干扰能力强、精度高
  • 通用性强，结构简单，便于集成
  • 具有"逻辑思维"能力

正逻辑与负逻辑

正逻辑：高电平用逻辑1表示，低电平用逻辑0表示
负逻辑：高电平用逻辑0表示，低电平用逻辑1表示

1.2 数制

常用数制

数制 基数 数码 进位规则
十进制 10 0-9 逢十进一
二进制 2 0, 1 逢二进一
八进制 8 0-7 逢八进一
十六进制 16 0-9, A-F 逢十六进一

数制转换

二进制→十进制：按权展开求和
例：(1011.101)₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ + 1×2⁻¹ + 0×2⁻² + 1×2⁻³ = 11.625
十进制→二进制：
整数部分：除2取余，逆序排列
小数部分：乘2取整，顺序排列
二进制↔十六进制：每4位二进制数对应1位十六进制数
二进制↔八进制：每3位二进制数对应1位八进制数

1.3 二进制数的算术运算

补码运算

正数的补码：与原码相同
负数的补码：符号位为1，数值位逐位求反再加1
补码加法：[A-B]补 = [A]补 + [-B]补
例：用4位补码计算 5-2
[5]补 = 0101，[-2]补 = 1110
[5-2]补 = 0101 + 1110 = 10011（舍弃进位）= 0011 = 3 ✓

溢出判断

溢出条件：两个同号数相加，结果符号与加数符号相反
判断方法：最高位进位与次高位进位异或为1则溢出

📖 第2章 逻辑代数基础

2.1 逻辑代数的基本定律

基本公式

名称 公式1 公式2
0-1律 A·0 = 0 A+1 = 1
自等律 A·1 = A A+0 = A
互补律 A·Ā = 0 A+Ā = 1
重叠律 A·A = A A+A = A
交换律 A·B = B·A A+B = B+A
结合律 (AB)C = A(BC) (A+B)+C = A+(B+C)
分配律 A(B+C) = AB+AC A+BC = (A+B)(A+C)
吸收律 A(A+B) = A A+AB = A
冗余律 AB+ĀC+BC = AB+ĀC

摩根定理（反演律）

摩根定理：
Ā·B̄ = A̅+B̅ （与非 = 非或）
A̅+B̄ = Ā·B̄ （或非 = 非与）

三个重要规则

代入规则：等式中的变量可用任意函数替代，等式仍成立
反演规则：求反函数时，与↔或，0↔1，原变量↔反变量
对偶规则：求对偶式时，与↔或，0↔1（变量不变）

2.2 逻辑函数的标准形式

最小项与最小项表达式

最小项定义：n个变量的乘积项，每个变量以原变量或反变量形式出现且仅出现一次
特点：
  • n个变量有2ⁿ个最小项
  • 每个最小项只有一组变量取值使其为1
  • 所有最小项之和为1
编号：使最小项为1的变量取值对应的十进制数

最大项与最大项表达式

最大项定义：n个变量的和项，每个变量以原变量或反变量形式出现且仅出现一次
编号：使最大项为0的变量取值对应的十进制数
关系：编号相同的最小项和最大项互为反函数

2.3 逻辑函数的化简

代数化简法

  • 并项法：利用 A+Ā=1 将两项合并
  • 吸收法：利用 A+AB=A 吸收多余项
  • 消去法：利用 A+ĀB=A+B 消去多余因子
  • 配项法：利用 A+Ā=1 增加项再化简

卡诺图化简法

卡诺图特点：
  • 几何相邻的最小项在逻辑上也相邻
  • 2ⁿ个相邻最小项合并可消去n个变量
  • 相邻包括：上下边相邻、左右边相邻、四角相邻
化简步骤：
1. 将逻辑函数写成最小项表达式
2. 在卡诺图中填入1（最小项）和0
3. 画包围圈（必须是2ⁿ个相邻的1）
4. 包围圈要尽可能大，数目尽可能少
5. 每个包围圈对应一个乘积项，相加即为最简式

无关项的利用

约束项：不允许出现的输入组合
任意项：输出可为0或1的输入组合
处理：在卡诺图中用×表示，根据化简需要取0或1

📖 第3章 逻辑门电路

3.1 MOS管的开关特性

NMOS管：
  • 当 vGS > VGS(th) 时导通，相当于开关闭合
  • 当 vGS < VGS(th) 时截止，相当于开关断开
PMOS管：
  • 当 |vGS| > |VGS(th)| 时导通
  • 当 |vGS| < |VGS(th)| 时截止

3.2 CMOS反相器

结构特点：PMOS管和NMOS管互补对称
工作原理：
  • 输入高电平：NMOS导通，PMOS截止，输出低电平
  • 输入低电平：PMOS导通，NMOS截止，输出高电平
重要特点：
  • 静态功耗几乎为零
  • 输入电阻高，带负载能力强
  • 开关速度快

3.3 CMOS逻辑门

CMOS与非门

结构：NMOS管串联（工作管），PMOS管并联（负载管）
逻辑：L = AB̄

CMOS或非门

结构：NMOS管并联（工作管），PMOS管串联（负载管）
逻辑：L = A̅+B̄

特殊输出结构

类型 特点 应用
漏极开路门(OD) 需外接电源和电阻，可实现线与 总线连接
三态门(TSL) 输出有高电平、低电平、高阻态三种状态 总线传输
传输门(TG) 双向模拟开关，可传输模拟信号 数据选择

3.4 TTL门电路

TTL反相器

电路结构：输入级、中间级、输出级
推拉式输出：提高开关速度和带负载能力

TTL与非门

输入级采用多发射极晶体管实现与功能
中间级和输出级实现非功能
逻辑：L = AB̄

集电极开路门(OC门)

特点：输出端只有一个三极管，需外接电源和电阻
应用：可实现线与功能

3.5 门电路的主要参数

参数 含义
VOH(min) 输出高电平的最小值
VOL(max) 输出低电平的最大值
VIH(min) 输入高电平的最小值
VIL(max) 输入低电平的最大值
VNH 高电平噪声容限 = VOH(min) - VIH(min)
VNL 低电平噪声容限 = VIL(max) - VOL(max)
扇出数 能驱动同类门的最大数目
传输延迟时间 输出波形相对输入波形的延迟
驱动条件：
1. VOH(min) ≥ VIH(min)
2. VOL(max) ≤ VIL(max)
3. IOH(max) ≥ IIH(total)
4. IOL(max) ≥ IIL(total)

📖 第4章 组合逻辑电路

4.1 组合逻辑电路的特点

定义：输出只取决于当前输入，与电路原来状态无关
结构特点：
  • 由门电路组成
  • 输出与输入之间没有反馈通路
  • 不含记忆元件

4.2 组合逻辑电路的分析

分析步骤：
1. 由逻辑图写出各输出端的逻辑表达式
2. 化简和变换逻辑表达式
3. 列出真值表
4. 根据真值表确定电路功能

4.3 组合逻辑电路的设计

设计步骤：
1. 逻辑抽象：确定输入、输出变量及逻辑状态含义
2. 列出真值表
3. 写出逻辑表达式
4. 化简和变换（根据器件类型）
5. 画出逻辑图

4.4 竞争与冒险

竞争：一个门的两个输入信号同时向相反方向变化
冒险：由于竞争引起输出出现错误的窄脉冲（毛刺）
产生条件：逻辑表达式可化为 A·Ā 或 A+Ā 形式

消除竞争冒险的方法

  • 加滤波电容：在输出端并联小电容（4~20pF）
  • 加选通信号：等待输入稳定后再取样
  • 增加冗余项：在卡诺图中包围圈相切处增加包围圈

📖 第5章 中规模组合逻辑集成电路

5.1 编码器

普通编码器

功能：将输入信号编成二进制代码
限制：任何时刻只允许输入一个有效编码信号
编码原则：N位二进制代码可表示2ᴺ个信号

优先编码器

特点：允许同时输入多个编码信号，按优先级编码
典型芯片：74LS148（8线-3线优先编码器）
使能端：EI（使能输入）、EO（使能输出）、GS（工作状态标志）

5.2 译码器

二进制译码器

功能：将n位二进制代码译成2ᴺ个输出信号
典型芯片：74LS138（3线-8线译码器）
输出：Yi = E·mi（E为使能，mi为最小项）
用译码器实现逻辑函数：
1. 将逻辑函数转换为最小项表达式
2. 将最小项对应的输出端接到与非门
3. 与非门输出即为逻辑函数

显示译码器

功能：将BCD码译成七段显示驱动信号
典型芯片：74HC4511、7448
显示器：LED数码管（共阴极/共阳极）

5.3 数据选择器（MUX）

功能：在地址选择信号控制下，从多路输入中选择一路输出
典型芯片：74HC151（8选1数据选择器）
输出：Y = Σ(Di·mi)，Di为数据输入，mi为最小项
用数据选择器实现逻辑函数：
方法一（N个变量）：变量接地址端，Di接0或1
方法二（N+1个变量）：变量接地址端，Di接0、1或第N+1个变量
方法三（≥N+2个变量）：需加辅助逻辑门

5.4 数据分配器（DEMUX）

功能：将一路输入数据分配到多个输出通道
实现：可用译码器实现（使能端接数据，地址端接选择信号）

5.5 数值比较器

功能：比较两个二进制数的大小
典型芯片：74LS85（4位数值比较器）
比较原则：从高位到低位依次比较

5.6 加法器

半加器与全加器

类型 输入 输出 表达式
半加器 A, B S, C S=A⊕B, C=AB
全加器 A, B, Ci S, Co S=A⊕B⊕Ci, Co=AB+(A⊕B)Ci

多位加法器

串行进位加法器：低位进位送给高位，速度慢
超前进位加法器：同时产生各位进位，速度快
典型芯片：74LS283（4位超前进位加法器）
加法器应用：
  • 实现减法运算：A-B = A + B补 = A + B反 + 1
  • 码制转换：8421码转余3码（加0011）
  • BCD码加法运算

📖 第6章 触发器

6.1 触发器的基本概念

定义：具有记忆功能的基本逻辑单元，能存储1位二进制信息
特点：有两个稳定状态（0态和1态），在触发信号作用下可转换

6.2 基本RS触发器

特性方程：Q* = S + R̄Q，约束条件：SR = 0
功能：
  • S=0, R=0：保持
  • S=1, R=0：置1
  • S=0, R=1：置0
  • S=1, R=1：不允许（约束条件）

6.3 同步触发器

同步RS触发器

特性方程：Q* = S + R̄Q（CP=1时有效）
约束条件：SR = 0

同步D触发器

特性方程：Q* = D（CP=1时有效）
特点：消除了约束条件

6.4 主从触发器

主从RS触发器：
  • CP=1时，主触发器接收信号，从触发器保持
  • CP=0时，主触发器保持，从触发器接收主触发器状态
  • 解决了一次变化问题

主从JK触发器

特性方程：Q* = JQ̄ + K̄Q
功能：
  • J=0, K=0：保持
  • J=1, K=0：置1
  • J=0, K=1：置0
  • J=1, K=1：翻转（计数）

6.5 边沿触发器

特点：只在时钟边沿（上升沿或下降沿）触发
优点：抗干扰能力强，不存在一次变化问题
类型：上升沿触发、下降沿触发

6.6 触发器的功能转换

目标触发器 转换方法 驱动方程
JK→D J=D, K=D̄ Q* = D
JK→T J=T, K=T Q* = T⊕Q
JK→T' J=1, K=1 Q* = Q̄
D→JK D = JQ̄ + K̄Q Q* = JQ̄ + K̄Q

📖 第7章 时序逻辑电路

7.1 时序逻辑电路的特点

定义：输出不仅取决于当前输入，还与电路原来状态有关
结构特点：
  • 包含存储电路（触发器）
  • 输出与输入之间存在反馈通路
描述方法：驱动方程、状态方程、输出方程、状态转换表、状态转换图、时序图

7.2 同步时序电路的分析

分析步骤：
1. 写出各触发器的驱动方程（激励方程）
2. 将驱动方程代入特性方程，得到状态方程
3. 写出输出方程
4. 列出状态转换表
5. 画出状态转换图和时序图
6. 分析电路功能

7.3 同步时序电路的设计

设计步骤：
1. 根据逻辑要求画出状态转换图
2. 状态化简（合并等价状态）
3. 状态编码（给每个状态分配二进制代码）
4. 列出状态转换表和驱动表
5. 求驱动方程和输出方程
6. 画出逻辑图
7. 检查自启动能力

7.4 计数器

同步计数器

特点：所有触发器使用同一时钟信号
分析方法：写出驱动方程→状态方程→状态转换表

异步计数器

特点：各触发器时钟信号不同
分析注意：需考虑时钟信号的有效边沿

集成计数器

芯片 功能 特点
74LS161 4位二进制同步计数器 同步预置、异步清零
74LS163 4位二进制同步计数器 同步预置、同步清零
74LS192 十进制可逆计数器 双时钟、异步预置
74LS290 二-五-十进制计数器 异步计数器
计数器级联：
N进制计数器级联可实现更大模值的计数
例：两个十进制计数器级联可实现100进制计数
实现任意模值计数：
方法一：反馈置零法（异步清零）
方法二：反馈预置法（同步预置）

7.5 寄存器和移位寄存器

寄存器

功能：存储二进制数据
典型芯片：74LS175（4位D触发器）

移位寄存器

功能：数据可逐位左移或右移
类型：
  • 单向移位寄存器（左移或右移）
  • 双向移位寄存器
  • 循环移位寄存器
典型芯片：74LS194（4位双向移位寄存器）
移位寄存器应用：
  • 串行-并行转换
  • 并行-串行转换
  • 序列信号发生器
  • 计数器

📖 第8章 时序逻辑集成电路

8.1 集成计数器的应用

构成任意进制计数器

反馈置零法：
N进制计数器在计数到N-1时产生置零信号
反馈预置法：
在计数到某个状态时产生预置信号，置入初始值
例：用74LS161实现十进制计数器
方法一（反馈置零）：计数到1001(9)时，下一个状态清零
方法二（反馈预置）：计数到1001时，预置为0000

计数器的级联

同步级联：所有计数器使用同一时钟，低位进位控制高位计数
异步级联：低位进位作为高位时钟

8.2 集成寄存器的应用

移位寄存器构成环形计数器：将移位寄存器的输出反馈到输入端，可构成环形计数器或扭环形计数器

📖 第9章 555定时器及多谐振荡器

9.1 555定时器

结构：由两个电压比较器、一个RS触发器和放电管组成
引脚功能：
  • TH（阈值输入）：高触发端
  • TR（触发输入）：低触发端
  • CO（控制电压）：外接控制电压
  • R̄（复位）：低电平有效
  • D（放电）：放电管输出
  • OUT（输出）：逻辑输出

9.2 555定时器的应用

单稳态触发器

功能：输入触发信号后，输出一个固定宽度的脉冲
脉冲宽度：tw = 1.1RC
应用：定时、延时、脉冲整形

多谐振荡器（无稳态触发器）

功能：产生矩形脉冲信号
振荡周期：T = (R1 + 2R2)C·ln2 ≈ 0.7(R1 + 2R2)C
占空比：q = (R1 + R2)/(R1 + 2R2)

施密特触发器

功能：将缓慢变化的信号转换为矩形波
特点：具有回差特性（两个不同的阈值电压）
应用：波形变换、脉冲整形、幅度鉴别

📖 第10章 存储器

10.1 存储器的分类

分类 类型 特点
按功能 ROM（只读存储器） 数据固定，断电不丢失
RAM（随机存取存储器） 可读可写，断电丢失
RAM类型 SRAM（静态RAM） 速度快，集成度低
DRAM（动态RAM） 需要刷新，集成度高
ROM类型 掩膜ROM 出厂时数据固定
PROM/EPROM/EEPROM 可编程/可擦除

10.2 存储器的扩展

位扩展：将多片存储器的数据线并联，地址线和控制线共用
字扩展：用高位地址线译码选择芯片，低位地址线共用

10.3 存储器实现组合逻辑函数

ROM实现逻辑函数：
地址线作为输入变量，数据线作为输出函数
每个地址单元存储对应的输出值

📖 第12章 D/A与A/D转换

12.1 D/A转换器（DAC）

主要参数

分辨率：输入数字量的位数，n位DAC有2ⁿ个输出电平
转换精度：实际输出与理想输出的偏差
转换速度：从输入数字量变化到输出稳定所需时间

常见DAC电路

  • 权电阻网络DAC：结构简单，但电阻值范围大
  • R-2R梯形电阻网络DAC：电阻种类少，便于集成
  • 权电流型DAC：速度快，精度高

12.2 A/D转换器（ADC）

A/D转换步骤

采样：将时间连续信号转换为时间离散信号
保持：在转换期间保持采样值不变
量化：将采样值转换为离散电平
编码：将量化后的电平用二进制代码表示
采样定理：fs ≥ 2fmax（采样频率大于信号最高频率的2倍）

常见ADC电路

类型 特点 应用
并行比较型（Flash） 速度最快，电路复杂 高速应用
逐次逼近型（SAR） 速度较快，精度较高 通用场合
双积分型 精度高，速度慢 数字万用表
Σ-Δ型 高分辨率，抗干扰强 音频处理

ADC主要参数

分辨率：输出数字量的位数
转换精度：实际转换值与理想值的偏差
转换时间：完成一次转换所需时间
量化误差：±0.5LSB（最小有效位）
