传感器概述

1.1 传感器的定义与组成

核心概念

传感器定义（GB7665—87）

能够感受规定的被测量并按照一定规律转换成可用输出信号的器件和装置。

传感器组成三要素

被测量 → [敏感元件] → [转换元件] → [基本电路] → 电量输出
  • 敏感元件：感受被测量，输出与被测量有确定关系的物理量
  • 转换元件：将敏感元件输出的非电物理量转换为电参量（R、C、L）
  • 基本电路：将电参量接入电路转换成电量（电压、电流）

核心部分是转换元件，决定传感器的工作原理。

传感器基本特性

2.1 静态特性指标

必考

静态特性：线性度、迟滞、重复性、灵敏度、分辨率、漂移、稳定性

（1）线性度

实际特性曲线与拟合直线的最大偏差程度。

γL = ±(ΔLmax / yFS) × 100%

拟合方法：理论拟合、端点连线拟合、最小二乘法拟合（精度最高）

（2）迟滞

正、反行程期间输出特性曲线不重合的现象。

γH = ±(ΔHmax / yFS) × 100%

产生原因：弹性元件滞后、铁磁体/铁电体的磁滞/电滞现象。

（3）重复性

同一方向多次测量时输出特性不一致的程度。

γR = ±(2~3)σmax / yFS × 100%
公式

（4）灵敏度

线性传感器：S = Δy / Δx （常数）
非线性传感器：S = dy / dx （变量）

单位：mV/mm（位移）、mV/℃（温度）等

（5）分辨率与阈值

  • 分辨率：传感器能检测到的最小输入增量
  • 阈值：输入小到某种程度输出不再变化的值

（6）漂移

输入量不变而输出量发生改变。包括：

  • 零点漂移与灵敏度漂移
  • 分为时间漂移（时漂）和温度漂移（温漂）

2.2 动态特性

核心

传递函数

H(s) = Y(s) / X(s) = (bmsm + ... + b0) / (ansn + ... + a0)

初始条件为零时，输出拉氏变换与输入拉氏变换之比。

一阶系统

H(s) = k / (τs + 1)
阶跃响应：y(t) = k(1 - e-t/τ)
频率响应：A(ω) = 1/√(1+ω²τ²)　　φ(ω) = -arctan(ωτ)

关键参数：时间常数 τ，τ越小响应越快，频带越宽。

t = τ时达到63.2%；t = 4τ时达到98.2%（工程上认为已稳定）。

二阶系统

H(s) = kωn2 / (s2 + 2ξωns + ωn2)

关键参数：固有频率 ωn、阻尼比 ξ

阻尼比响应特性
ξ = 0零阻尼，等幅振荡
ξ < 1欠阻尼，衰减振荡
ξ = 1临界阻尼，响应时间最短
ξ > 1过阻尼，稳定时间较长

实际取 ξ = 0.6～0.8，过冲量不太大，稳定时间不太长。

ωn ≥ (3～5)ω（信号频率），避免共振。

2.3 误差理论基础

误差分类

类型定义消除方法
系统误差服从确定规律的误差修正值法、对称法、替代法、对照法
随机误差随机变化的误差多次测量取平均值
粗大误差人为疏忽引起的显著误差拉依达准则（3σ准则）剔除
公式

相对误差类型

实际相对误差：γA = Δx / A × 100%
示值相对误差：γx = Δx / x × 100%
满度相对误差：γn = Δx / yFS × 100%（用于确定精度等级）
重点

贝塞尔公式

σ̂ = √[Σvi2 / (n-1)]

用残差近似代替随机误差求标准差估计值。测量次数一般取 n = 10～20。

电阻式传感器

3.1 金属丝电阻应变片

核心原理

电阻应变效应

导体在外力作用下产生机械形变时阻值发生变化。

ΔR/R = Δl/l - ΔS/S + Δρ/ρ = (1 + 2μ + λE)ε = k0ε

其中 k0 ≈ 1 + 2μ ≈ 1.5～2（金属应变片）

应变片结构

  • 敏感栅：高阻金属丝/金属箔
  • 基片：绝缘材料
  • 盖片：保护层

类型：丝式、箔式（最常用）、薄膜型

主要特性

  • 横向效应：圆弧部分使灵敏系数下降，箔式应变片可减小此误差
  • 机械滞后：加载和卸载时残余应变未释放
  • 蠕变：恒力作用下应变电阻值随时间变化
  • 温度误差：环境温度变化引起的附加电阻变化

3.2 测量电路——电桥

必考

直流电桥

平衡条件：R1·R3 = R2·R4
桥路类型工作臂数输出电压灵敏度特点
单臂桥 1个 Uo = KεUi/4 S = Ui/4 输出小，有非线性
半桥（差动） 2个 Uo = KεUi/2 S = Ui/2 灵敏度翻倍，非线性改善
全桥（差动） 4个 Uo = KεUi S = Ui 灵敏度最高，应尽量采用
关键结论
  • 差动测量可消除非线性误差
  • 相邻桥臂应变极性一致→输出为两者之差；极性不同→输出为两者之和
  • 供桥电压越高灵敏度越高，但温度误差越大（一般1～3V）

温度补偿方法

  • 桥路补偿：相邻臂接补偿片（不受力，同温度场）
  • 差动桥自补偿：各臂同温度场，温度变化相减为零
  • 应变片自补偿：特殊材料使温度附加应变为零

3.3 应用

  • 力传感器：柱式、悬臂梁式（等截面/等强度）、轮辐式
  • 压力传感器：膜片式（径向应变εr、切向应变εt）
  • 加速度传感器：悬臂梁+质量块，频率范围10～60Hz
  • 电子皮带秤：Q = p × v，总量 = ∫Q dt

3.4 半导体压阻式传感器

对比

金属应变片 vs 半导体应变片

特性金属应变片半导体应变片
灵敏系数k ≈ 1.5～2k = πE ≈ 100～200
主要变化几何尺寸变化电阻率变化（压阻效应）
温度稳定性较好差
非线性小大应变时严重

电容式传感器

4.1 工作原理与类型

基本公式
C = εS/δ = ε0εrS/δ

改变 S → 变面积型；改变 δ → 变极距型；改变 ε → 变介质型

电容式传感器特点

  • 输出阻抗高，功率小
  • 极板静电引力小，可动质量小
  • 固有频率高，动态响应好
  • 可进行非接触测量

4.2 输出特性

重点

变极距型（δ）

单极板：ΔC/C0 = (Δδ/δ0)[1 + (Δδ/δ0) + (Δδ/δ0)² + ...]
灵敏度：k0 = 1/δ0
差动式：ΔC/C0 = 2(Δδ/δ0)　灵敏度提高一倍
差动非线性误差：γL ≈ (Δδ/δ0)² × 100%

灵敏度与初始极距δ0成反比，δ0小灵敏度高但测量范围小。

变面积型（S）

ΔC/C0 = -Δa/a　（线性关系）
灵敏度：k0 = εb/δ = 常数

输出特性为线性，适合大位移测量。

变介质型（ε）

应用：测厚度、测位移、测液位（油量测量）、测温湿度

4.3 测量电路

主要电路类型

  • 变压器式交流电桥：差动电容接两桥臂，输出与位移成线性
  • 二极管双T型电路：URL = MUEf(C1-C2)
  • 运算放大器式电路：Uo = -UiC0δ/(εs)，解决非线性问题
  • 调频电路：电容变化→频率变化，需鉴频器

4.4 应用

  • 电容式压力传感器：膜片式差动结构，ΔC ∝ ΔP
  • 电容测厚仪：金属带材轧制厚度在线检测
  • 电容式加速度计：质量块+差动电容
  • 电容传声器：驻极体电容传声器（麦克风）
  • 电容式料位计：利用介电常数变化检测物位

电感式传感器

5.1 变磁阻式（自感式）传感器

核心公式
L = N²μ0S0 / (2δ0)

改变气隙厚度δ → 变气隙型（灵敏度高，非线性严重）

改变截面积S → 变面积型（线性好，灵敏度低）

差动式优势
  • 灵敏度提高一倍
  • 非线性减小（多乘Δδ/δ因子，消除偶次项）
  • 温度自补偿

5.2 差动变压器式（互感式）传感器

工作原理

初级线圈加交流激励，两个次级线圈差动连接。铁芯位移改变互感M1、M2，输出电压反映位移大小和方向。

Uo = ω(M2 - M1)Ui / √(R12 + (ωL1)2)
考点

零点残余电压

铁芯在中间位置时，理论上输出应为零，实际Uo ≠ 0。

原因：两次级线圈电气参数不完全对称、电源高次谐波、寄生电容。

消除方法：串联电阻、并联电容、相敏检波电路。

5.3 电涡流式传感器

核心

电涡流效应

金属导体置于变化磁场中，导体内部产生闭合电流（涡流）。

形成条件：①存在交变磁场 ②金属导体处于交变磁场中

等效阻抗

Z1 = f(ρ, μ, x, ω)
被测量（位移x、材料ρ/μ）→ 阻抗变化 → 电信号

高频反射式：测位移、振动、转速、探伤

低频透射式：测金属板厚度

应用
  • 非接触测位移、振动、转速
  • 金属板厚度测量
  • 电涡流探伤（表面裂纹检测）
  • 轴心轨迹测量（两个90°探头）
  • 转速测量：n = 60f/n齿

磁电/磁敏式传感器

6.1 霍尔效应

核心公式

霍尔电势

UH = KH·I·B
KH = RH/d = ρμ/d （霍尔灵敏度）
RH = 1/(ne) （霍尔常数）

霍尔电势与电流I和磁场B的乘积成正比。

霍尔元件材料选择

  • 金属：电子浓度n高→RH很小→灵敏度低 ✗
  • 绝缘体：电阻率ρ大但迁移率μ很小 ✗
  • 半导体：ρ较大、μ适中 → 最适合作霍尔元件 ✓
  • 多采用N型半导体（电子迁移率大于空穴）
  • 厚度d越小灵敏度越高（d ≈ 1μm）

6.2 误差与补偿

不等位电势

B=0时UH≠0的现象。原因：引出电极安装不对称、材料不均匀。

补偿：等效为电桥，在电阻大的桥臂并联电阻。

温度误差补偿

恒流源供电 + 并联分流电阻Rp（可选负温度系数）

6.3 应用

📏 测位移

在梯度磁场中，UH取决于位置

⚡ 测电流

B∝Ix，UH = KHIcKBIx

🔄 测转速

n = 60×f/n齿

🚗 汽车点火

霍尔式无触点电子点火装置

霍尔集成传感器

  • 线性型：输出电压与B成线性关系，用于测位移、振动、磁场
  • 开关型：高/低电平转换，有回差特性，用于测转速、无触点开关

压电式传感器

7.1 压电效应

核心概念

正压电效应与逆压电效应

  • 正压电效应：外力→变形→内部极化→表面产生电荷（力→电）
  • 逆压电效应：施加电场→电介质形变（电→力）

压电效应是可逆的，压电元件可实现机电双向转换。

7.2 压电材料

石英晶体（单晶体）

  • 化学式 SiO2，六面体结构
  • X轴（电轴）：纵向压电效应，qxx = dxxFx
  • Y轴（机械轴）：横向压电效应
  • Z轴（光轴）：不产生压电效应
  • dxx = 2.31×10-12 C/N

压电陶瓷（多晶体）

  • 钛酸钡(BaTiO3)、锆钛酸铅(PZT)
  • 需人工极化后才具有压电效应
  • 压电常数大，灵敏度高
  • 稳定性不如石英晶体

7.3 测量电路

关键

压电传感器等效电路

  • 压电元件可等效为电荷源（并联电容）或电压源（串联电容）
  • 输出阻抗高，必须配接高输入阻抗的前置放大器
  • 两种前置放大器：电压放大器和电荷放大器

电荷放大器（推荐）

输出电压与传感器电荷成正比，不受电缆电容影响。

7.4 应用

  • 动态力测量：机械冲击、振动
  • 压电加速度计：质量块+压电元件
  • 超声波换能器：利用逆压电效应发射超声波
  • 压电传声器：声电转换
  • 点火装置：燃气灶、打火机

光电传感器

8.1 光电效应

三种光电效应
类型原理器件
外光电效应光照射→电子逸出物体表面光电管、光电倍增管
内光电效应光照→半导体电导率增加光敏电阻
光生伏特效应光照→PN结产生电动势光电池、光电二极管

8.2 光电器件

光敏电阻

内光电效应，光照→电阻减小，灵敏度高，响应慢

光电二极管

光生伏特效应，反向工作，光电流与照度成正比

光电三极管

有放大作用，灵敏度比光电二极管高

光电池

大面积PN结，不需外加电源，短路电流与照度成正比

8.3 应用

  • 光电转速计：光脉冲计数
  • 光电编码器：角位移/角速度测量
  • 光纤传感器：利用光纤传光特性测量
  • CCD图像传感器：图像采集
  • 光栅传感器：精密位移测量

新型光电传感器

9.1 光纤传感器

光纤传光原理

基于光的全反射原理。当入射角大于临界角时，光在纤芯与包层界面发生全反射，实现光的传输。

数值孔径 NA = sinθmax = √(n12 - n22)

光纤传感器分类

  • 功能型（FF型）：光纤本身就是敏感元件
  • 非功能型（NF型）：光纤仅作传光用，另加敏感元件

9.2 红外传感器

红外辐射特性

  • 波长范围：0.75～1000μm
  • 热效应显著
  • 应用：测温、热成像、气体分析、遥控

9.3 超声波传感器

工作原理

  • 利用压电效应实现电-声转换
  • 频率 > 20kHz
  • 应用：测距、测厚、探伤、液位测量
  • 测距公式：d = v·t/2（v为声速，t为传播时间）
